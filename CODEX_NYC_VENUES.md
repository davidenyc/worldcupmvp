# Codex Data Sprint — NYC Venue Population

**Read this entire file before writing any code.**

Goal: bring NYC to 140+ real, named venues so the app feels complete and ready to show to anyone. Right now there are ~40 curated real venues + auto-generated placeholders. After this sprint, every pin on the NYC map should be a real place someone can walk into.

---

## Editorial philosophy — READ THIS FIRST

Every venue must earn its place. Apply this test before adding anything:

**For major countries (Mexico, France, England, Spain, Brazil, Argentina, Portugal, Germany, etc.):**
- The venue must actually show games on screens. No exceptions.
- `note` must mention screens, TVs, projectors, or watch-party atmosphere explicitly.
- `showsSoccer: true`, `numberOfScreens >= 4` preferred, `gameDayScore >= 8.0`.
- Fine-dining restaurants without a proper bar and dedicated screens do NOT belong here — even if they are culturally associated with that country.

**For smaller/obscure countries (Senegal, Ghana, Iran, Ecuador, etc.):**
- It is OK to include 1–2 cultural community restaurants where that country's expats gather.
- These must be tagged `venueIntent: "cultural_dining"`, `showsSoccer: false` (or `true` only if they genuinely turn on matches), and `gameDayScore <= 7.5`.
- The `note` must be honest: say it's a cultural hub or that the community gathers here, NOT that it's a watch-party spot.

**Every note must contain at least one of these signals for any non-cultural-dining venue:**
- screen count ("14 screens", "multiple screens", "a projector")
- explicit watch reference ("watch party", "shows all World Cup matches", "opens at 6am for kickoffs")
- atmosphere signal ("packed for every match", "vocal crowd", "match-day energy")

  return {
    id: `sports-bar-${index}`,
    slug: `nyc-sports-bar-${slugify(seed.name)}`,
    name: seed.name,
    address: seed.address,
    borough: seed.borough,
    neighborhood: seed.neighborhood,
    postalCode: seed.postalCode,
    lat: seed.lat,
    lng: seed.lng,
    venueTypes: seed.venueTypes,
    venueIntent: "sports_bar",
    likelySupporterCountry: null,
    associatedCountries: seed.associatedCountries,
    showsSoccer: true,
    openNow: false,
    rating: seed.rating,
    reviewCount: Math.floor(seed.rating * 180 + index * 40),
    priceLevel: seed.priceLevel,
    approximateCapacity: seed.approximateCapacity,
    capacityBucket:
      seed.approximateCapacity < 30 ? "under_30"
      : seed.approximateCapacity < 60 ? "30_60"
      : seed.approximateCapacity < 100 ? "60_100"
      : seed.approximateCapacity < 200 ? "100_200"
      : "200_plus",
    capacityConfidence: seed.capacityConfidence,
    reservationType: seed.reservationType,
    acceptsReservations: seed.reservationType !== "none",
    reservationUrl: seed.reservationUrl,
    reservationPhone: seed.reservationPhone,
    website: seed.website,
    instagramUrl: seed.instagramUrl,
    imageUrls: [`https://picsum.photos/seed/sb${index}/800/500`],
    numberOfScreens: seed.numberOfScreens,
    hasProjector: seed.hasProjector,
    hasOutdoorViewing: seed.hasOutdoorViewing,
    familyFriendly: seed.familyFriendly,
    standingRoomFriendly: seed.standingRoomFriendly,
    goodForGroups: seed.goodForGroups,
    privateEventsAvailable: seed.privateEventsAvailable,
    atmosphereTags: seed.atmosphereTags,
    cuisineTags: seed.cuisineTags,
    supporterNotes: `A dedicated soccer bar in ${seed.neighborhood} — multiple screens, match-day energy, and a mixed international crowd.`,
    matchdayNotes: seed.note,
    editorialNotes: seed.note,
    gameDayScore: seed.gameDayScore,
    fanLikelihoodScore: seed.gameDayScore * 0.9,
    rankScore: seed.gameDayScore * seed.editorialBoost,
    rankingReasons: ["Dedicated soccer bar", "Multiple screens", "Match-day atmosphere"],
    isFeatured: seed.editorialBoost >= 1.2,
    isOfficialFanHub: false,
    verificationStatus: "demo_editorial",
    sourceName: "editorial",
    sourceConfidence: seed.sourceConfidence,
    reservationType2: seed.reservationType,
    isRealVenue: true,
    city: "nyc",
  } as unknown as Venue;
}
